Hârtie

a minimal HTML presentation engine

For navigation, click for the next slide, or use the arrow keys.

About

Hârtie was developed because most of the HTML presentation frameworks (at least at the time of creation), were quite bloated.

In contrast, Hârtie is minimalistic; it does only the strictest of functions, with no fancy transitions, or anything too complicated for someone who's never fiddled with HTML.

Don't get me wrong, it still is powerful, but its power is inherited from HTML.

Functions overview

  • Slide navigation1:
    • next slide = click or right arrow key
    • previous slide = left arrow key
    • home slide = home key
    • end slide = end key

  • That's it.
    • 1 this is the default behaviour.

How to use Hârtie

First, include hartie.js in your to-be-presentation. Then, specify the behaviour of the page via the onload, onclick and onkeydown events applied to the window. You now have a presentation going!

You can optionally specify a stylesheet to suit your needs.

Slides should be marked as belonging to the slide class, like this:

<div class="slide">
  ... your slide here ...
</div>

That's it. You should also see the source of this page for a living, breathing example!

Misc

You can add a footer to the page like this:

<div id="footer">footer</div>
And it'll show up like this!
